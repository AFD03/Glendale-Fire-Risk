Glendale Fire Risk Visualization

Interactive Wildfire Risk Assessment for Glendale, California

üöß Coming Soon

This project visualizes wildfire risk in Glendale, California using GIS data, a weighted risk model, and an interactive Plotly Dash web dashboard. It aims to help city planners, emergency services, and residents understand spatial patterns of fire vulnerability at the building level.

Project Features

  • Interactive 2D Plotly Dash dashboard showing building-level fire risk
  • Embedded 3D QGIS scene with terrain, extruded buildings, and risk overlay
  • Full data pipeline documentation and final report
  • Comparison with CalFire Fire Hazard Severity Zones (FHSZ)

UC Davis STA160 - Team 3

A collaborative project by students in Data Science and Statistics, combining GIS analysis, risk modeling, and interactive web visualization to address wildfire risk assessment.
